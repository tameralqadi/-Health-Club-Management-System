# The Health Club Management System (Software Engineering Project)
A comprehensive Software Engineering project focused on automating the admission, management, and booking processes within gyms and fitness clubs. Built using the **Oracle APEX** framework, this system replaces manual administrative tasks with a secure, centralized online platform.

## Software Engineering Highlights
- Requirements Engineering: Translated real-world gym operational challenges into clear functional requirements for multiple user classes.
- System Architecture: Implemented a robust Role-Based Access Control (RBAC) model to ensure strict data isolation and security boundaries.
- Database Design: Structured relational data models to handle complex transactional flows like payment history and live scheduling.

## System Actors & Use Cases
### 1. Admin (System Administrator)
- Package Lifecycle Management: Complete CRUD operations on membership packages, including dynamic cost configuration and discounts.
- Member & Trainer Directory: Full control over adding, updating, and deleting system users.
- Business Intelligence: Generates financial reports and analyzes member attendance statistics.
- Resource Scheduling: Defines and modifies available training session slots.

### 2. Member (End User)
- Self-Service Portal: Accesses profile information, tracks active subscriptions, and reviews secure payment history.
- Dynamic Booking: Explores available trainers and books real-time training sessions.
- Electronic Payments: Securely processes membership fees.

### 3. Trainer
- Schedule Management: Monitors and reviews booked training sessions scheduled by members.
- Attendance Tracking: Records and updates daily gym member attendance logs.

## Tech Stack & Environment
- **Platform/Framework:** Oracle APEX (Application Express)
- **Database Engine:** Oracle SQL & PL/SQL (Relational Schema Design)
- **Security Control:** Role-Based Access Control (RBAC)
